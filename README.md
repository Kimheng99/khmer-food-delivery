# Food Delivery Website

A full-stack food delivery web application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

This project allows users to browse food menus, add items to cart, place orders, and checkout.
The admin dashboard allows administrators to manage food items and update delivery order status.

---

# Features

## User Features

* View food menu
* Add food to cart
* Remove items from cart
* Fill delivery information
* Checkout and place orders
* View order details

## Admin Features

* Upload new food items
* View food list
* Manage customer orders
* Update delivery status

# Installation

## Clone Repository

```bash
git clone https://github.com/your-username/food-delivery.git
```

---

# Backend Setup

Go to server folder:

```bash
cd server
```

Install dependencies:

```bash
npm install
```

Create `.env` file:

```env
MONGO_URL="mongodb+srv://kimhengfood:Kimheng9@cluster0.wwsd2.mongodb.net/EC?appName=Cluster0"
JWT_SECRET=code123
PORT=5000
```

Run backend server:

```bash
npm run server
```

---

# Frontend Setup

Go to frontend folder:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Run frontend:

```bash
npm run dev
```
